# 东南大学自动定时抢讲座脚本说明

本脚本基于 [seu-lecture-reserve](https://github.com/404874351/seu-lecture-reserve) 的思路，实现了东南大学讲座的自动定时预约功能，目前已帮助多位同学完成讲座任务。

## 使用方法

1. 前往 [超级鹰](https://www.chaojiying.com/) 注册并获取账号信息。

2. 在项目根目录下的 `config.json` 文件中填入个人信息，如下示例：

   ```json
   {
     "loginSEU": {
       "id": "<一卡通号>",
       "pwd": "<一卡通密码>"
     },
     "chaojiying": {
       "user": "<超级鹰账号>",
       "pass": "<超级鹰密码>",
       "softid": "<超级鹰软件ID>",
       "codetype": 1004
     }
   }
   ```

3. 运行 `main.py` ，输入要预约的讲座序号。

## 注意事项

### 配置相关

- 脚本默认使用 **谷歌浏览器（Chrome）**，如需使用其他浏览器，请在 `main.py` 中自行修改浏览器配置。
- 脚本默认生成 **playwright_user_data** 文件用于保存浏览器用户数据。
- 请勿缺失 `brotli` 依赖。

### 运行相关

- 首次运行或其他情况下，可能需要输入 **手机短信验证码**，如输入不及时，可将 `main.py` 中 `get_cookie_auto` 函数的 `timeout` 参数调大，以延长等待时间。
- 尽量在预约开始时间前 **十分钟** 内运行，且使用前 **同步系统时间** 以提高成功率。
